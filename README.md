### 🦊 关于我 | About Me

你好，我是 **Vulpara**。

一名栖息在二进制缝隙里的**灵狐**，目前正在**计算机科学**的丛林里进行第二年的巡游。我不满足于枯燥的代码堆砌，更偏爱将**理性的算法**与**感性的美学**编织在一起。

在我的世界里，代码不仅是运行的指令，更是撩拨数字世界的指尖戏法。

### 🪄 灵感与技艺 | Tech Stack

我擅长在不同的语言间切换姿态，捕捉那一抹转瞬即逝的灵感：

* **坚韧如骨 (C++)**：用最冷硬的逻辑，搭建算法的基石。在 `Pointer` 与 `Memory` 的博弈中，寻找极致的掌控感。
* **柔若无骨 (Python)**：链接神经网络的深邃梦境。在 `AI` 与 `Data` 的流动中，窥见未来的私语。
* **流光溢彩 (Front-end)**：这是我挥洒妩媚色彩的画布。使用 `Vue/React` 与 `CSS` 编织交互，让冰冷的网页在呼吸间溢出灵动之美。
